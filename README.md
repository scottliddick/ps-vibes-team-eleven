## Roshambo Royale - Battle Royale Rock-Paper-Scissors

A unique twist on the classic rock-paper-scissors game featuring:

- **4-player tournament bracket** with semifinals and finals
- **Ghost mechanics** - Eliminated players become ghosts who can curse living players
- **Hot-seat multiplayer** - Pass the device around for privacy
- **Beautiful UI** with gradient backgrounds and smooth animations

### How to Play

1. Enter 4 player names
2. Two semifinal matches determine the finalists
3. Losers become ghosts who can force players to throw specific moves
4. The final match crowns the Roshambo Royale champion!

### Running the Game

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS

A battle royale twist on rock-paper-scissors! 4 players compete in a tournament bracket, but losing doesn't mean you're out - you become a **ghost** who can curse the living players and force their moves!

### Features

- **4-Player Tournament**: Single elimination bracket with semifinals and finals
- **Ghost Mechanics**: Losers become ghosts who can curse living players before each match
- **Hot-Seat Multiplayer**: Pass the device around - privacy screens prevent peeking
- **Beautiful UI**: Gradient effects, smooth transitions, and emoji-based moves
- **Tie Resolution**: Automatic rematches when players throw the same move

### How to Play

1. Enter 4 player names
2. View the tournament bracket
3. In each match:
   - Ghosts (if any) curse a player by forcing their move
   - Players take turns choosing rock, paper, or scissors
   - Winner advances, loser becomes a ghost
4. Finals: The last two players face off with all ghosts cursing
5. Champion is crowned!

### Running the Game

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Tech Stack

- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- React 19
